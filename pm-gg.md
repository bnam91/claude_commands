## GG 전용 스크립트

| 스크립트 | 설명 |
|----------|------|
| `_db_get_today.js` | 브리핑 DB에서 오늘 페이지 찾거나 생성. PAGE_ID / START_TOGGLE_ID / END_TOGGLE_ID 출력 |
| `_eod_read.js` | 마감브리핑용. 현빈02 콜아웃 + 지혜 업무코멘트 읽어서 출력 |
| `db_read.js` | 프로젝트 DB 전체 조회 (계층 구조) |
| `dday_count.js` | 런칭 페이지 D-XX 갱신 |
| `db_goal_status.js` | GOAL 행 상태 업무막힘 → 진행 중 변경 |
| `db_add_task.js` | DB에 업무 추가 (`--parent "키워드"` 옵션) |
| `db_set_friday_deadline.js` | depth 1 + 우선순위 1 항목 데드라인을 이번 주 금요일로 일괄 설정 |

## GG 전용 요청 트리거

| 사용자 요청 | 실행할 명령 |

| "이번 주 데드라인" / "주간 마감 설정" | `node db_set_friday_deadline.js` |
