## Phase 구조

각 Phase는 목표가 있고, 달성 시 다음 Phase로 넘어간다.
Phase 전환 시 목표 점검 + 다음 Phase 계획 수립.

| Phase | 목표 | 완료 기준 | 상태 |
|-------|------|-----------|------|
| Phase 1 | 채널 가동 + 영상 10편 업로드 | 10편 완료 + 편집자 채용 완료 | 진행 중 |
| Phase 2 | 구독자 성장 + 첫 수익 | (논의 예정) | 대기 |
| Phase 3 | 연매출 2억~2.5억 달성 | 수익화 강의 30명 + 프로그램 200명 + 출강 발생 | 대기 |

## 콘텐츠 제작 프로세스 (영상 1편 기준)

소재 확정 → 주제 확정 → 플로우(개요) 작성 → 스크립트 작성 → 필수 키컷 구상 및 사전 배치 → 촬영 (화면 녹화 + 보이스오버) → 편집 작업 → 편집 완료 + 검토 → 썸네일 제작 → 업로드 + SEO 세팅

> 콘텐츠는 수익화 강의 커리큘럼 챕터 기반으로 기획 — 영상 하나하나가 강의의 무료 챕터로 기능해야 한다 (일이관지).
