## 견적서 생성 절차

사용자가 "견적서 만들어줘", "견적서 복사해줘", "견적서 생성해줘" 등을 요청하면:

### Step 1. 정보 수집

아래 항목을 사용자 메시지에서 파악. 없는 항목은 물어봐.

| 항목 | 설명 | 필수 |
|------|------|------|
| 거래처명 | 수신자 파악용 (파일명 미사용) | 선택 |
| 견적일 | 날짜 (없으면 오늘 기본) | 선택 |
| 수신 | 예: `팔도 (윤종혁 선임님)` | 선택 |
| 담당자 | 예: `신현빈` | 선택 |
| 상품명 | 예: `쿠팡리뷰체험단` | 선택 |
| 품목 목록 | 품명, 수량, 단가 (여러 개 가능) | 선택 |

품목 정보가 없어도 일단 시트를 만들고 나중에 직접 입력할 수 있어.

### Step 2. 코멘트 확인

품목 정보를 다 받은 후, 기본 코멘트 추가 여부 물어봐:

> "아래 기본 코멘트를 추가할까요?
>
> *쿠팡정책상 회원등급에 따른 리뷰어 별 상품구매 가격이 상이함으로 상품가액 및 배송비는 작업완료 후 실비정산 되었습니다. (리뷰어별 실결제비 액셀첨부)*"

- "응 / 추가해줘 / 기본으로" → `--comment "기본"` 사용
- "아니 / 필요없어" → `--comment` 생략
- 직접 코멘트 입력 → `--comment "입력내용"` 사용

### Step 3. 스크립트 실행

수집한 정보로 명령어 구성:

```bash
cd ~/Documents/claude_skills/estimate && \
python3 estimate.py --create \
  --name "[거래처명]" \
  [--date YYYY-MM-DD] \                # 파일명: Quotation_goya_YYMMDD
  [--date-kr "YY년 M월 D일"] \
  [--recipient "수신자"] \
  [--manager "담당자"] \
  [--product "상품명"] \
  [--items '[{"name":"품목명","qty":수량,"price":단가}]'] \
  [--comment "기본" or "직접입력"]
```

출력 형식: `OK|파일ID|파일명|URL`

### Step 4. 결과 보고

```
✅ 견적서 생성 완료

  파일명 : [파일명]
  링크   : [URL]

링크를 클릭하면 바로 편집할 수 있어요.
```

## 시트 구조 참고

| 셀 | 내용 |
|----|------|
| C3 | `견적일 : YY년 M월 D일` |
| C4 | `수 신 : 수신자` |
| C5 | `담 당 : 담당자` |
| C6 | `상 품 : 상품명` |
| ROW11~28 | 품목 (C=품명, G=수량, I=단가) → 공급가액/세액은 자동수식 |
| B32 | 코멘트 (병합셀) |

|-----------|------|
