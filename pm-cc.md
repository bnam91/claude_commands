## 애자일 DB 구조 규칙

- **단위**: 상품 1개 = 루트 1행 (`🛍 N차 상품`)
- **삽입 방식**: 역순 입력 (Notion은 최신 생성이 위로 표시되므로)
- **관계**: `상위 항목` relation (single_property, 자기 참조)
- **`하위 항목` 칼럼 없음** — 사용하지 않음

**상품 1개당 구조**
```
🛍 N차 상품
  └ ✅ 시장 발굴 및 기획
      └ 골드박스 셀러 모니터링
      └ 쿠팡 탑텐 키워드 필터링 (소거법 → 10~20개)
      └ 네이버 데이터랩 트렌드 확인
      └ ★키워드 수집 시트 입력 + 흥미점수 배정
      └ 후보 키워드 5개로 컷
      └ 키워드별 탑셀러 3명 판매량 조회 (아이템스카우트)
      └ 마진 계산 (원가 + 쿠팡수수료 10.8% + 배송비)
      └ 최종 진입 키워드 선정 (GO/NO-GO)
      └ 탑셀러 점수 배정 (가격/제품/디자인/상세페이지 1~5점)
      └ 진입 전략 확정 (① 가격 ② 성능 ③ 디자인 ④ 마케팅)
      └ 최종 GO 선언 및 타겟 상품 확정
  └ ✅ 상품 소싱
      └ 경쟁사 상품 주문 (실제 사용 경험)
      └ 1688 거래처 찾기
      └ 샘플 소싱 (배대지 경유)
      └ 샘플 검수 (품질·포장·리드타임 확인)
      └ 본품 발주 확정 (수량·단가·납기 협의)
  └ ✅ 상품 런칭 준비 (상세페이지)
      └ 상품 기획 (포지셔닝 전략 반영)
      └ 판매페이지 제작
      └ 상품 썸네일 제작
  └ ✅ 상품 입고
      └ 쿠팡 로켓그로스 입고 (바코드 출력 → 중국 전달 → 배대지)
      └ 국내 3PL 창고 입고
  └ ✅ 상품 등록
      └ 쿠팡 상품 등록
      └ 리뷰체험단 세팅
  └ ✅ 마케팅
      └ 노마진 마케팅 세팅
      └ 쿠팡광고 집행
      └ 카페 핫딜 게시
```
