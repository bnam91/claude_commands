pm-xx(역전야매치트키 유튜브 채널, Phase 1 진행 중)가 이번주 weekly meeting 토글에 PM 브리핑을 작성하는 스킬이야.

## 프로젝트 컨텍스트

| 항목 | 내용 |
|------|------|
| 프로젝트명 | XX (역전야매치트키) |
| 연간 목표 | 연매출 2억~2.5억 (수익화 강의 + 자체 프로그램 + 출강) |
| 현재 Phase | Phase 1 — 채널 가동 + 영상 10편 업로드 |
| 완료 기준 | 10편 완료 + 편집자 채용 완료 |
| 타겟 | 5인 미만 사업자, 프리랜서, 예비 핵개인 |
| 콘텐츠 방향 | 수익화 강의 커리큘럼 챕터 기반 영상 (얼굴 없는 채널) |

## 실행

```bash
node ~/Documents/claude_skills/agent_meeting/agent_pm_input.js --pm pm-xx --type weekly
```

## 동작 방식

1. 이번주 weekly meeting 페이지 자동 탐색
2. "pm-xx — 이번주 목표" 토글에 내용 덮어쓰기
3. pm-xx DB 자동조회 미구현 → `--content`로 직접 작성 권장

## 직접 내용 입력 (권장)

```bash
node ~/Documents/claude_skills/agent_meeting/agent_pm_input.js \
  --pm pm-xx --type weekly \
  --content "이번주 목표: 영상 1편 촬영 완료\n블로킹: 스크립트 미완\n현빈 지원요청: 촬영 시간 확보"
```

## 브리핑 작성 가이드

주간 브리핑에 포함할 항목:
- **이번주 집중 목표**: Phase 1 기준 이번주 완료할 콘텐츠 단계 (소재→스크립트→촬영→업로드)
- **현재 Phase 진행률**: 10편 중 몇 편 완료/진행 중/예정
- **블로킹 이슈**: 편집자 미채용, 스크립트 지연 등
- **현빈 지원 요청**: 촬영 시간 배정, 주제 최종 결정 등 현빈이 직접 해야 할 것
- **PM 코멘트**: D-day 대비 현재 속도 평가 + Phase 1 완료 예상 시점

## 사용자 요청 처리

- "pm-xx 주간 브리핑 입력해줘" / "pm-xx 이번주 목표 써줘" → 실행
